# India-General-Election-2024-Dashboard
A dynamic and interactive data visualization dashboard built to analyze India General Election 2024 results, focusing on alliance performance, state-wise comparisons, constituency analysis, and election trends.

## Purpose

The India General Election Results 2024 Dashboard is a visually engaging and analytical Power BI report designed to help users explore and compare election outcomes across states and constituencies. The dashboard focuses on highlighting alliance performance, candidate insights, vote distribution, and key election metrics.

## Tech Stack

The dashboard was built using the following tools and technologies:

• 📊 Power BI Desktop – Main data visualization platform used for report creation.  
• 🔄 Power Query – Data transformation and cleaning layer for preparing data.  
• 📈 DAX (Data Analysis Expressions) – Used for calculated measures and KPIs.  
• 🗂️ Data Modeling – Relationships established among multiple tables for analysis.  
• 📁 File Format – .pbix for development and .png for dashboard previews.  

## Data Source

Source: India General Election Results 2024 Dataset

The dataset includes state-wise and constituency-wise election data containing candidate information, party names, alliance details, vote counts, vote shares, and winning margins.

## Features / Highlights

### • Business Problem

Analyzing large election datasets manually makes it difficult to identify voting trends, compare alliance performance, and generate meaningful insights.

### • Goal of the Dashboard

- Analyze NDA, I.N.D.I.A., and Other alliance performance  
- Compare state-wise election results  
- Track constituency-level insights  
- Visualize candidate performance and vote trends  

### • Walk Through of Key Visuals

• Overview Analysis – Displays total seats and alliance performance  
• State Demographic Analysis – State-wise election insights  
• Political Landscape by State – Party-wise seat distribution  
• Constituency Analysis – Candidate and vote analysis  
• Detailed Grid View – Exportable election details  
• Interactive Filters & Drill-through Reports  

### • Business Impact & Insights

• Provides faster election trend analysis  
• Helps compare alliance performance efficiently  
• Supports data-driven insights and decision-making  
• Improves accessibility and visualization of election data
